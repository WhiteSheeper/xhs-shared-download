# 小红书分享图片下载

#### 一个简单的根据小红书分享链接获取图片的项目,加了一个页面方便使用

本来是加了点击下载图片功能的,但是后来觉得一般都是在手机上使用,就去掉了改为长按保存了,网页的话就右键图片保存吧

docker部署的话拉去下来进入文件夹内直接docker compose up -d ,默认项目端口是10010
