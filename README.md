# 小红书图片下载工具

一个简单易用的小红书图片批量下载工具，支持去水印，适配移动端和桌面端。

## 功能特点

- 🖼️ 支持批量下载小红书图片，自动去除水印
- 📱 完美适配移动端和桌面端
- 🚀 简单易用，粘贴分享文本即可使用
- 💫 支持长按（移动端）/ 右键（桌面端）保存图片
- ⚡ 响应式布局，图片网格自适应显示

## 在线体验

访问地址：[xhs.dashuaibi.monster](https://xhs.dashuaibi.monster)

## 使用说明

1. 打开小红书APP，点击分享按钮
2. 选择"复制链接"
3. 将复制的内容粘贴到输入框
4. 点击获取图片
5. 长按(手机)或右键(电脑)保存图片
